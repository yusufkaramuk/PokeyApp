# PokeyApp v0.3.0 ✨ İlk Kararlı MVP Sürümü (Initial Release)

Sohbet geçmişindeki geliştirme planının **Faz 6** adlı nihai aşamasının tamamlanmasıyla birlikte, **PokeyApp projesinin indirilebilir ve kullanılabilir ilk resmi paketi** oluşturulmuştur!

### 📥 İndirme ve Kurulum
Aşağıdaki **`PokeyApp-v0.3.0-win-x64.zip`** dosyasını indirin. Klasöre çıkarıp içindeki `install.ps1`'e sağ tıklayıp *Yönetici Olarak Çalıştır* diyerek saniyeler içinde bilgisayarınıza güvenle kurun (Güvenlik / Firewall kayıtları otomatik atılır).

### 🚀 Neler Var?
- [x] **Tek Tuşla Dürtme:** Eski MSN Messenger tarzı, dikkat bozmayan titreşim bildirimleri (WPF Native).
- [x] **Otomatik Keşif:** Manuel IP girmeye gerek kalmadan ağı tarayan UDP altyapısı.
- [x] **Gelişmiş Ağ Hata Ayıklama:** Bağlantı koptuğunda artan intervallerle (exponential backoff) sistemi yormadan kendini onaran soket yapısı.
- [x] **Gizlilik & Odak (Focus-safe):** Tam ekran bir uygulamadayken pencerenizin aktifliğini ve odağını bozmayan özel Windows modları kullanıldı.
- [x] **Hızlı Kurulum Betiği:** Hedef kurulum için 1 tıklamayla kısayollar ve tray eklentisi.
