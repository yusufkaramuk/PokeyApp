# PokeyApp v0.3.0 ✨

İlk kararlı (pre-v1) sürüm yayınlandı! 🎉

### 🚀 Neler Var?
- [x] **Tek Tuşla Dürtme:** Eski MSN Messenger tarzı, dikkat bozmayan titreşim/dürtme bildirimleri.
- [x] **Otomatik Keşif:** Manuel IP girmeye gerek kalmadan ağı otomatik tarayan UDP alt yapısı.
- [x] **Akıllı Ağ Bağlantısı:** Bağlantı koptuğunda saniye bazında artarak kendini yeniden toplayabilen TCP/IP soket mantığı.
- [x] **Gizlilik & Odak (Focus-safe):** Tam ekran oyun veya özel iş yaparken dahi asla ana pencerenizin odağını bozmayan, arka planda bildirim veren altyapı.
- [x] **Gece Modu UI:** Yepyeni modern Windows tasarımlarına uygun Dark-mode (Gece) ekranı.
- [x] **Hızlı Kurulum (Install Script):** Tek tıklamayla Windows güvenlik duvarı (Firewall) üzerinden gerekli izinleri vererek bilgisayara gizlice kuran güçlü powershell eklentisi.

### 📥 Nasıl İndirip Kurulur?
Aşağıdan **`PokeyApp-v0.3.0-win-x64.zip`** dosyasını indirin (aşağıdaki Assets bölümünden).
Klasörün içine çıkarın, içinde yer alan **`install.ps1`** dosyasına sağ tıklayıp "PowerShell ile Yönetici Görevinde Çalıştır" deyin. Tüm port ve kısa yollar otomatik çekilecektir. İyi eğlenceler!
